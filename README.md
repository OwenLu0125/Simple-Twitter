## 功能說明

前台：<br/>
使用者可以註冊/登入帳號、推文、回覆推文、設定帳號資料、編輯個人資訊、跟隨/取消跟隨其他使用者、查看單一偏貼文及該篇文章的回覆內容。<br/>

後台：<br/>
管理者可以登入帳號、觀看推文清單及使用者列表、刪除推文。

##  使用說明

      1.  確認已經安裝 node.js 與 npm
      2.  clone 專案
      3.  終端機進入資料夾並輸入：`$ npm `
      4.  安裝完畢後，繼續輸入：`$ npm run start`
